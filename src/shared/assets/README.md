## 🏳️ Flag Icons

SVG flag icons used by the language switcher in the header:

| File | Language |
|------|----------|
| `flags/gb.svg` | English (United Kingdom) |
| `flags/es.svg` | Espanol (Spain) |
| `flags/fr.svg` | Francais (France) |
| `flags/cn.svg` | Chinese (China) |

These are imported by the `LanguageSwitcher` widget and displayed alongside locale labels in the language dropdown.
